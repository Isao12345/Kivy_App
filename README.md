# โปรแกรมจัดการข้อมูลห้องเรียน (Kivy Application)

## ภาพรวมของโปรแกรม

โปรแกรมนี้พัฒนาด้วยภาษา Python และใช้ไลบรารี Kivy
เป็นแอปพลิเคชันแบบหลายหน้า (Multi-Screen Application)

โปรแกรมใช้สำหรับจัดการข้อมูลห้องเรียน กิจกรรม และปฏิทิน
โดยแบ่งการทำงานออกเป็นหลายหน้าชัดเจน



## โครงสร้างหน้าจอของโปรแกรม

โปรแกรมประกอบด้วย 3 หน้าหลัก:

1. หน้า MyClass
   - จัดการข้อมูลหรือรูปภาพประจำห้องเรียน
   - แสดงและบันทึกข้อมูล

2. หน้า Event
   - ใช้จัดการงานหรือกิจกรรมต่าง ๆ Done สำหรับกิจกรรมที่เสร็จแล้ว และ Delete สำหรับลบหน้ากิจกรรม
   - แสดงรายละเอียดของกิจกรรม เช่น Days left และ สถานะของกิจกรรม

3. หน้า Calendar
   - แสดงปฏิทิน
   - ใช้ดูวันที่และเพิ่มกิจกรรม

การเปลี่ยนหน้าจอใช้ ScreenManager ของ Kivy
ทำให้สามารถสลับไปมาระหว่างหน้าต่าง ๆ ได้

## หลักการทำงานโดยสรุป

- เมื่อรันไฟล์ `main.py` โปรแกรมจะสร้าง App และโหลดทุกหน้าจอ
- ระบบจะแสดงหน้าแรกของโปรแกรม
- ผู้ใช้สามารถเปลี่ยนหน้าไปยัง MyClass, Event หรือ Calendar ได้
- ข้อมูลที่บันทึกจะถูกเก็บไว้ในไฟล์ภายในโฟลเดอร์ `data`
- โปรแกรมสามารถอ่านและเขียนข้อมูลด้วยไฟล์ JSON

## วิธีการติดตั้งและใช้งานโปรแกรม
1. การดึงโปรเจกต์จาก GitHub

ผู้ใช้งานต้องติดตั้งโปรแกรม Git ให้เรียบร้อยก่อน จากนั้นเปิดโปรแกรม Command Prompt หรือ PowerShell แล้วใช้คำสั่งต่อไปนี้เพื่อทำการคัดลอก (Clone) โปรเจกต์จาก GitHub:

git clone https://github.com/Isao12345/Kivy_App.git


เมื่อดำเนินการเสร็จสิ้น ให้เข้าสู่โฟลเดอร์ของโปรเจกต์ด้วยคำสั่ง:

cd REPOSITORY_NAME


หากต้องการอัปเดตโค้ดให้เป็นเวอร์ชันล่าสุด สามารถใช้คำสั่ง:

git pull origin main

2. การติดตั้งไลบรารีที่จำเป็น (Dependencies)

เพื่อป้องกันปัญหาความขัดแย้งของไลบรารี แนะนำให้สร้าง Virtual Environment ก่อนใช้งาน โดยดำเนินการดังนี้:

สร้าง Virtual Environment:

python -m venv venv


เปิดใช้งาน Virtual Environment:

สำหรับ Windows:

venv\Scripts\activate


สำหรับ macOS / Linux:

source venv/bin/activate


จากนั้นติดตั้งไลบรารีที่ระบุไว้ในไฟล์ requirements.txt ด้วยคำสั่ง:

pip install -r requirements.txt


ระบบจะทำการติดตั้งแพ็กเกจทั้งหมดที่จำเป็นต่อการทำงานของโปรแกรมโดยอัตโนมัติ

3. การรันโปรแกรม

เมื่อดำเนินการติดตั้งครบถ้วนแล้ว สามารถรันโปรแกรมได้โดยใช้คำสั่ง:

python main.py


หากไม่มีข้อผิดพลาด โปรแกรมจะเริ่มทำงานตามที่ออกแบบไว้
